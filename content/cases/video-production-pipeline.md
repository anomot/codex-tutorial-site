# 场景
- 把一个主题做成短视频或教程视频。
- 需要脚本、镜头、素材和发布检查一起完成。

# 前置条件
- 知道目标时长和受众。
- 能准备或生成相关素材。
- 有基本的导出和预览流程。

# 真实操作流程
1. 先生成脚本和镜头表。
2. 准备素材并检查叙事顺序。
3. 导出预览后检查字幕、节奏和同步。
4. 确认封面、标题和简介可发布。

# 核心观点
- 视频工作流的难点是阶段多且依赖强。
- 每一阶段都要单独验证。

# 可复用知识
- 脚本母版可以重复使用。
- 镜头表比口头描述更容易执行。

# 最佳实践
- 先做短片段，再做完整视频。
- 把字幕、音画同步和封面一起审。
- 保留脚本、镜头和发布清单。

# 常见坑
- 脚本太长，镜头不可执行。
- 没有检查字幕同步。
- 素材风格不统一。

# 测试与验证方法
- 预览视频时长和节奏是否合适。
- 检查字幕、旁白和镜头切换。
- 确认导出格式适合平台。
